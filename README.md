# Wisetracker SDK integration with VUE

## Project Setup

```sh
yarn
```

## Project Includes

1. Wisetracker SDK init
2. logScreen (Routing)
3. LogEvent (login, search)

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
